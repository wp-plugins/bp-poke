BuddyPress Poke Plugin allow members to poke users just like facebook.

== Description ==

BuddyPress Poke Plugin allows members to poke other members.

= How it works:- =

User visits other users profile and uses poke button.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.
1. Download the zip file and extract
1. Upload `bp-poke` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu.
1. Enjoy

== Frequently Asked Questions ==

= Does This plugin works without BuddyPress =
No, It needs you to have BuddyPress Installed and activated
